Shortcodes & Usage
==================

Available Shortcodes
---------------------

WP Mautic currently provides a shortcode API through ``shortcodes.php``.

Example Usage
~~~~~~~~~~~~~

::

  [mautic type="form" id="1"]

Parameters
----------

- **type:** Type of Mautic asset (``form``, etc.).
- **id:** The ID of the Mautic form or asset you want to embed.
